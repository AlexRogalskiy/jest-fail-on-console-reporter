## 🌩 Installation

```
npm install --save-dev jest jest-fail-on-console-reporter
```

## 🕹 Usage

Replace the default reporter with jest-fail-on-console-reporter in
jest.config.json:

```js
"reporters": [
  "./node_modules/jest-fail-on-console-reporter"
]
```
